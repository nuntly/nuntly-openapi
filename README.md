# Nuntly OpenAPI Specification

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)

OpenAPI 3.1 specification for the [Nuntly](https://nuntly.com) REST API.

[Documentation](https://nuntly.com/docs) | [API Reference](https://nuntly.com/docs/api) | [Get your API key](https://app.nuntly.com/signup)

## Files

| File | Description | Usage |
|------|-------------|-------|
| `openapi.yaml` | Technical spec (55 operations) | Postman, Insomnia, code generators, API testing |
| `openapi.documented.yaml` | Spec with field examples | Mintlify docs, Redoc, developer portal |

## Usage

### Import in Postman

1. Open Postman
2. File > Import
3. Select `openapi.yaml`

### Import in Bruno

1. Open [Bruno](https://www.usebruno.com/)
2. File > Open Collection > Import OpenAPI V3 Spec
3. Select `openapi.yaml`
